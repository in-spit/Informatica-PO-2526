Servers


Een server is een pc die speciaal gemaakt is om
programma's te hosten op het internet,
of om grote hoeveelheden data op te slaan.
servers zijn een groot deel van de reden dat jij
elke dag op tiktok of youtube content kan kijken.
servers draaien de hele dag door om daarvoor te zorgen.

 

Verschillende soorten servers:

Home server

file server

AI data center


Home servers

Een home server is een computer die speciaal ingesteld is
om op een thuisnetwerk bestanden en programma's te hosten,
waardoor je niet aan 1 apparaat vast zit voor je bestanden.
deze servers worden pre-made op de markt aangeboden, maar
veel mensen maken nu een server van hun ongebruikte tech.
dit kan met besturingssystemen zoals Ubuntu server of Fedora Server OS.

om hier meer over te leren kan je kijken naar deze besturingssystemen
